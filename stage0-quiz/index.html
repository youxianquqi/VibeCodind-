阶段 0 · HTML/CSS 练习（教材式）

设计原则：先讲清再动手。每课页面上有「是什么 / 合法取值 / 常见误区」，再让你在 Cursor 里填代码。不要跳课。

怎么用

  1. 浏览器打开某一课的 .html，把讲解读完。
  2. Cursor 打开该课指定的文件，只改标了 TODO 的地方。
  3. 保存后刷新浏览器，对照「自检」。
  4. 卡住再看 solutions/，不要先抄。

判据三题在第 5、7 课：Flex 居中、Flex 两栏、响应式导航。前面课是地基。

课程顺序（按这个走）

  8. 08 综合页 · 合在一起默写一遍
