03 · 定位 position

先记住「相对谁移动」，再填五种英文关键字。不要猜拼写。

五种合法值（只能从这里选）

  • static — 默认。在正常文档流里，top/left 无效。
  • relative — 仍占原来的位，视觉上相对自己原来的位置偏移。
  • absolute — 脱离文档流，相对最近的定位祖先（position 不是 static 的祖先）定位；没有则相对初始包含块。
  • fixed — 相对视口（浏览器窗口）定位，滚动时常钉在屏幕上。
  • sticky — 先按正常流，滚到阈值后像粘住（相对最近滚动祖先）。

「定位祖先」= 自己的祖先里，position 为 relative / absolute / fixed / sticky 的那个。 只写 relative 常用来当 absolute 孩子的参照。

不要发明第六种值。拼写必须全小写英文。
