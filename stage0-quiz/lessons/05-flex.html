05 · Flex：居中 + 两栏

阶段 0 判据：不查文档写出这两段。先读「主轴/交叉轴」，再写 CSS。

最小概念集（够用）

  • display: flex — 把该元素变成弹性容器，直接子元素变成弹性项。
  • 默认主轴是水平（一行排），交叉轴是垂直。
  • justify-content — 管主轴对齐（水平居中常用 center）。
  • align-items — 管交叉轴对齐（垂直居中常用 center）。
  • flex: 1 — 这项「吃掉剩余空间」（两栏主区常用）。
  • flex-shrink: 0 — 不要被挤扁（固定宽侧栏常用）。

水平 + 垂直居中公式（背下来）： display:flex; justify-content:center; align-items:center;

参考： MDN Flexbox · Flexbox Froggy
