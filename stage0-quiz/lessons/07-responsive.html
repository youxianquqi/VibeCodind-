07 · 响应式：viewport + 导航

先理解「为什么需要 meta」，再写媒体查询。缩小浏览器窗口做自检。

viewport 是干什么的

手机浏览器默认常按「桌面宽度」缩放页面，字会很小。 加了 viewport meta 后，布局宽度 = 设备宽度，才谈得上响应式。 

<meta name="viewport" content="width=device-width, initial-scale=1" />

这一行写在 <head> 里，不是写在 CSS 里。

媒体查询（合法条件示例）

「视口宽度至少 768px」写成：

@media (min-width: 768px) {
  /* 桌面样式 */
}

移动优先思路：默认写手机样式 → 用 min-width 加大屏样式。

响应式导航思路（本课要写的）

  1. 默认（窄屏）：.nav-links { display: none; } 隐藏链接
  2. 宽屏：@media (min-width: 768px) 里改成 display: flex 横排

完整汉堡按钮要点 JS；判据只要求「显隐 + 横排」这段 CSS。
