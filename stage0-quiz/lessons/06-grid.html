06 · Grid：列、fr、gap

Grid 适合二维格子。本课只练「定义列」和「间距」。

最小概念集

  • display: grid — 网格容器
  • grid-template-columns — 有几列、每列多宽
  • fr — 按份分剩余空间（fraction）
  • repeat(3, 1fr) — 等价于 1fr 1fr 1fr
  • gap — 格子之间的间距

左固定右自适应常见写法：grid-template-columns: 200px 1fr;

参考： MDN Grid · Grid Garden
